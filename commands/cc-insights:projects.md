# cc-insights 프로젝트별 분석

프로젝트별 Claude Code 사용 패턴을 분석합니다.

## 사용법

### 프로젝트 목록 및 통계 (기본)

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/analyzer.py" projects --days 14
```

### 특정 프로젝트 상세 분석

`$ARGUMENTS`에 `--project <프로젝트명>`이 포함되어 있으면:

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/analyzer.py" projects --project <프로젝트명> --days 14
```

## 분석 항목

### 프로젝트 목록 보기

- 프로젝트별 프롬프트 수
- 프로젝트별 세션 수
- 활동량 비율 시각화

### 프로젝트 상세 분석

- 총 프롬프트 수 및 기간
- 세션 수 및 세션당 평균 프롬프트
- 평균 품질 점수
- 모호한 프롬프트 비율
- 피크 작업 시간대
- 프로젝트별 Skill 자동화 후보
- 개선 필요 프롬프트 예시
