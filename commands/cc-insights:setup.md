# cc-insights 초기 설정 (온보딩)

최초 설치 시 실행하여 전체 히스토리를 분석하고 개인화된 인사이트를 제공합니다.

## 실행

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/analyzer.py" --setup
```

## 분석 항목

### 1. 개발 프로필 생성
- **작업 스타일**: 얼리버드/나이트아울/심야형/균형형
- **세션 스타일**: 딥다이브/스프린터/밸런서
- **프로젝트 스타일**: 집중형/포커스형/멀티태스커
- **프롬프트 스타일**: 상세파/간결파/적정파

### 2. 강점/약점 분석
- 프롬프트 품질 평가
- 명확성 분석
- 워크플로우 일관성
- 비효율 패턴 감지

### 3. Skill 자동화 추천
- 반복 패턴 감지 (3회 이상)
- 신뢰도 기반 우선순위
- 바로 생성 가능한 명령어 제공

### 4. 액션 아이템
- 우선순위별 구체적인 다음 단계
- 바로 실행 가능한 명령어 포함

### 5. 주요 프로젝트 현황
- 프로젝트별 활동량 시각화
- 작업 시간 패턴 분석

## baseline 저장

분석 결과가 `${CLAUDE_PLUGIN_ROOT}/data/baseline.json`에 저장됩니다.

이후 모든 분석은 이 baseline과 비교하여 변화를 추적합니다.

## 참고

- baseline은 언제든 `/cc-insights:setup`을 다시 실행하여 갱신할 수 있습니다.
- 분석에 시간이 걸릴 수 있습니다 (히스토리 크기에 따라 5-30초).
