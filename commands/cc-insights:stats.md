# cc-insights 전체 사용 통계

Claude Code 전체 사용 히스토리를 분석하여 통계를 제공합니다.

## 실행

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/analyzer.py" stats
```

## 분석 항목

### 1. 사용 기간
- 첫 사용 시간
- 마지막 사용 시간
- 총 사용 기간 (일)

### 2. 활동 통계
- 총 프롬프트 수
- 총 프로젝트 수
- 총 세션 수
- 하루 평균 프롬프트/세션

### 3. 토큰 사용량 (추정)
- 입력 토큰 (프롬프트 길이 기반)
- 출력 토큰 (입력의 약 7배로 추정)
- 총 토큰

### 4. 작업 패턴
- 가장 활발한 시간대
- 가장 활발한 요일

### 5. 월별 활동
- 월별 프롬프트 수
- 월별 세션 수
- 월별 프로젝트 수

### 6. 프로젝트 목록
- 전체 프로젝트 목록

## 참고

- 토큰 사용량은 프롬프트 길이를 기반으로 한 **추정치**입니다
- 실제 토큰 사용량은 Anthropic 콘솔에서 확인하세요
- 전체 히스토리를 분석하므로 시간이 다소 걸릴 수 있습니다
