# cc-insights 개인화된 팁

당신의 사용 패턴을 분석하여 맞춤형 개선 팁을 제공합니다.

## 실행

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/analyzer.py" tips --days 14
```

## 팁 생성 기준

### 1. 작업 스타일 기반
- 야간 작업이 많으면 → handoff 관련 팁
- 긴 세션이 많으면 → /compact 관련 팁

### 2. 프롬프트 스타일 기반
- 짧은 프롬프트가 많으면 → 구체화 팁
- 지시대명사 사용이 많으면 → 명확화 팁

### 3. 비효율 패턴 기반
- /clear 과다 사용 → claude --continue 팁
- 동시 세션 과다 → /rename 팁

### 4. 강점/약점 기반
- 약점 영역에 대한 구체적 개선 제안

## 출력 예시

```
╔══════════════════════════════════════════════════════════════╗
║  💡 개인화된 팁                                               ║
╚══════════════════════════════════════════════════════════════╝

  🔴 [세션 관리]
     긴 세션 중간에 `/compact`로 컨텍스트 압축을 권장합니다
     └─ 이유: 토큰 절약 및 응답 품질 유지

  🔴 [프롬프트 품질]
     프롬프트에 '기대 결과'와 '제약 조건'을 추가해보세요
     └─ 이유: 더 정확한 응답을 받을 수 있습니다

  🟡 [작업 패턴]
     야간 작업 후에는 `/handoff`로 컨텍스트 정리를 권장합니다
     └─ 이유: 다음 날 작업 재개 시 시간 절약
```

## 우선순위

- 🔴 높음: 즉시 적용 권장
- 🟡 중간: 점진적 개선
- 🟢 낮음: 선택적 적용
