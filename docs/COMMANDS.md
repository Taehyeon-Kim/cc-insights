# cc-insights 명령어 가이드

## 명령어 목록

### /cc-insights:setup

초기 설정을 수행하고 baseline을 생성합니다.

```bash
/cc-insights:setup
```

**기능:**
- 전체 히스토리 스캔
- 사용자 프로필 생성 (작업 스타일, 세션 패턴 등)
- 강점/약점 분석
- baseline 데이터 저장 (향후 비교용)
- 자동화 추천 skill 목록 생성
- 액션 아이템 제안

**출력 예시:**
```
╔══════════════════════════════════════════════════════════════════════════╗
║                    🔍 cc-insights 온보딩 분석 완료                         ║
╠══════════════════════════════════════════════════════════════════════════╣
║  📊 분석 기간: 2024-11-15 ~ 2025-01-14 (60일)
║  📝 총 프롬프트: 7,500개
║  📈 하루 평균: 125개
║  ⭐ 종합 등급: A
╚══════════════════════════════════════════════════════════════════════════╝

📋 당신의 개발 프로필
  • 작업 스타일: 🌙 나이트 아울 (저녁 집중형)
  • 세션 스타일: 🔥 딥다이브 (긴 세션 선호)
  • 프로젝트 스타일: 🌐 멀티태스커 (다중 프로젝트)
  • 프롬프트 스타일: ✍️ 적정파 (적절한 길이)
```

### /cc-insights:summary

빠른 현황 요약을 제공합니다. 매일 확인하기 좋습니다.

```bash
/cc-insights:summary
```

**옵션:**
- `--days N`: 분석 기간 지정 (기본: 7일)

**출력 예시:**
```
┌─────────────────────────────────────────────────────────────┐
│  📊 cc-insights 빠른 요약 (최근 7일)                         │
├─────────────────────────────────────────────────────────────┤
│  오늘: 45개 (📈 평균 대비 +20%)
│  총 프롬프트: 312개 | 품질: 8.5/10
│  모호한 프롬프트: 32개 (10.3%)
└─────────────────────────────────────────────────────────────┘

💡 추천: /commit skill 생성 (23회 반복)
```

### /cc-insights:analyze

상세 분석 리포트를 생성합니다.

```bash
/cc-insights:analyze
/cc-insights:analyze --days 30
/cc-insights:analyze report  # 마크다운 파일로 저장
```

**옵션:**
- `--days N`: 분석 기간 지정 (기본: 7일)
- `report`: 리포트를 `reports/` 폴더에 저장

**분석 항목:**
1. 프롬프트 품질 점수 분포
2. 반복 패턴 → Skill 제안
3. 비효율 패턴 감지
4. 세션 관리 패턴
5. 시간/요일별 활동 패턴
6. 프로젝트별 활동

### /cc-insights:tips

사용자 패턴 기반 개인화된 팁을 제공합니다.

```bash
/cc-insights:tips
/cc-insights:tips --days 14
```

**팁 유형:**
- 🔴 높은 우선순위: 즉시 적용 권장
- 🟡 중간 우선순위: 점진적 개선
- 🟢 낮은 우선순위: 선택적 적용

**출력 예시:**
```
╔══════════════════════════════════════════════════════════════╗
║  💡 개인화된 팁                                               ║
╚══════════════════════════════════════════════════════════════╝

  🔴 [/clear 과다 사용]
     `claude --continue` 사용으로 컨텍스트 유지
     └─ 이유: 82회 사용

  🔴 [프롬프트 길이]
     컨텍스트, 예상 결과, 제약 조건 포함 권장
     └─ 이유: 15자 미만 프롬프트 33%

  🟡 [작업 패턴]
     야간 작업 후에는 `/handoff`로 컨텍스트 정리를 권장합니다
     └─ 이유: 다음 날 작업 재개 시 시간 절약
```

### /cc-insights:trends

주간 트렌드를 분석합니다.

```bash
/cc-insights:trends
/cc-insights:trends --weeks 8
```

**옵션:**
- `--weeks N`: 분석할 주 수 (기본: 4주)

**분석 항목:**
- 품질 점수 변화 추이
- 모호한 프롬프트 비율 변화
- 사용량 변화

**출력 예시:**
```
╔══════════════════════════════════════════════════════════════╗
║  📈 최근 4주 트렌드                                           ║
╚══════════════════════════════════════════════════════════════╝

  품질 점수: 📈 상승
  모호한 프롬프트: 📈 개선
  사용량: ➡️ 유지

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    프롬프트: [████████████████████] 2252개
    품질: 9.0/10 | 모호: 11.1%
```

### /cc-insights:skills

자동화 가능한 skill 목록을 보여주고 생성할 수 있습니다.

```bash
/cc-insights:skills
/cc-insights:skills generate commit
```

**분석 기준:**
- 3회 이상 반복된 패턴 감지
- 신뢰도 점수 계산

**출력 예시:**
```
╔══════════════════════════════════════════════════════════════╗
║  🤖 Skill 자동화 추천                                         ║
╚══════════════════════════════════════════════════════════════╝

  1. /commit - 변경사항 커밋
     반복 횟수: 150회
     신뢰도: [██████████] 95%
     샘플:
       • "커밋해줘"
       • "변경사항 커밋"

  2. /handoff - 핸드오프 문서 자동 작성
     반복 횟수: 65회
     신뢰도: [████████░░] 82%
```

### /cc-insights:projects

프로젝트별 사용 패턴을 분석합니다.

```bash
/cc-insights:projects
/cc-insights:projects --project my-project
/cc-insights:projects --days 30
```

**옵션:**
- `--project <name>`: 특정 프로젝트 상세 분석
- `--days N`: 분석 기간 지정

**프로젝트 목록 출력:**
```
╔══════════════════════════════════════════════════════════════╗
║  📁 프로젝트별 분석 (최근 14일)                               ║
╚══════════════════════════════════════════════════════════════╝

  총 24개 프로젝트, 2,827개 프롬프트

   1. alphaswarm       [████████████████████] 1051개 (37.2%)
      └─ 세션: 146개
   2. laytonalpha      [██████████████████░░]  530개 (18.7%)
      └─ 세션: 78개
```

**프로젝트 상세 출력:**
```
╔══════════════════════════════════════════════════════════════╗
║  📁 프로젝트 상세 분석: alphaswarm                            ║
╚══════════════════════════════════════════════════════════════╝

  📊 기본 정보

  • 기간: 2026-01-04 ~ 2026-01-12 (9일)

  ⭐ 품질 분석
  • 평균 품질 점수: 9.2/10

  ⏰ 작업 시간대
```

## 권장 사용 흐름

```
1. 최초 설정
   └─ /cc-insights:setup

2. 매일 확인
   └─ /cc-insights:summary

3. 주간 리뷰
   ├─ /cc-insights:analyze
   ├─ /cc-insights:trends
   └─ /cc-insights:tips

4. 월간 리뷰
   ├─ /cc-insights:stats
   └─ /cc-insights:projects
```
